# Texte de la fiche App Store — français

## Identité

- **Nom de l’application** : Stockpile
- **Catégorie principale** : Flux de produits
- **Catégorie secondaire** : Importateur de données de boutique

## Proposition de valeur

Stockpile transforme les feuilles de calcul et les documents fournisseurs en
produits Shopify révisables, accompagnés de leurs sources.

## Description courte

Transformez les fichiers fournisseurs en ébauches de produits traçables,
vérifiez chaque modification, puis publiez le catalogue approuvé sur Shopify.

## Description

Stockpile aide les marchands à transférer les données produit de feuilles de
calcul et de documents fournisseurs vers Shopify sans copier-coller manuel.

1. Téléversez un fichier pris en charge depuis l’application intégrée.
2. Révisez les données produit extraites avant toute écriture dans Shopify.
3. Ouvrez un lien source pour vérifier la cellule ou la section justificative.
4. Publiez les produits approuvés et confirmez le résultat dans Shopify Admin.

Fonctions incluses :

- extraction guidée de fichiers produit;
- révision appuyée par les sources avant la création des produits;
- suggestions sur la santé du catalogue;
- facturation gérée par Shopify;
- messages de reprise clairs pour l’authentification, le traitement et la facturation.

Les prix, la période d’essai et les frais d’utilisation doivent être saisis
uniquement dans les champs de tarification désignés de la fiche.
